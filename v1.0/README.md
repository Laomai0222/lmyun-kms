## 使用方法
* ###### 首先下载LMYUN-KMS_v1.0.zip，解压后打开LMYUN-KMS_v1.0.exe
* ###### 进入后会先检测与服务器的连接，若无法连接，则退出
* ###### 进入首页后，可输入命令进行操作
* ##### 常用命令：
* ###### Auto (先激活Windows，再激活office)
* ###### AWin (只激活Windows)                                       
* ###### AO (只激活office)                                          
* ###### Set (设置KMS服务器)                                          
* ###### List (列出参考的KMS服务器列表)
